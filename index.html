What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web content using elements and tags.

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content goes here
  </body>
</html>

Common HTML Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <div> - Container/Division
  • <span> - Inline container
  • <a> - Links
  • <img> - Images
  • <ul>, <ol>, <li> - Lists

HTML Attributes

Attributes provide additional information about elements:

<img src="image.jpg" alt="description">
<a href="https://example.com">Link Text</a>

Semantic HTML

Semantic elements clearly describe their meaning:

  • <header> - Header section
  • <nav> - Navigation links
  • <main> - Main content
  • <article> - Independent content
  • <footer> - Footer section

Forms

Forms are used to collect user input:

<form>
  <input type="text">
  <input type="submit">
</form>

Best Practices

  • Always declare DOCTYPE
  • Use semantic elements when possible
  • Include proper meta tags
  • Make your HTML accessible
  • Validate your HTML code

What is Git Bash?

Git Bash is a terminal emulator that provides a Git command line experience on Windows. It includes common bash commands and Git-specific commands.

Essential Git Bash Commands

$ pwd

Print Working Directory - Shows current location

$ ls

List files and directories

$ cd [directory]

Change Directory

$ mkdir [directory]

Create new directory

$ touch [filename]

Create empty file

Basic Git Commands

$ git init

Initialize a new Git repository

$ git clone [url]

Clone a repository

$ git add [file]

Add file to staging area

$ git commit -m "message"

Commit staged changes

$ git push

Push commits to remote repository

$ git pull

Pull changes from remote repository

Git Branch Operations

$ git branch

List all branches

$ git branch [branch-name]

Create new branch

$ git checkout [branch-name]

Switch to specified branch

$ git merge [branch-name]

Merge specified branch into current branch

Git Status and History

$ git status

Show working tree status

$ git log

Show commit history

$ git diff

Show changes between commits

Git Configuration

$ git config --global user.name "[name]"

Set global username

$ git config --global user.email "[email]"

Set global email

$ git config --list

List all configurations

HTML Audio Elements

The HTML <audio> element is used to embed sound content in documents.

Sample Audio Player

Credit: NASA Public Domain Audio

Audio Element Code Example:

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Audio Attributes:

  • controls - Displays audio controls (play/pause, volume)
  • autoplay - Automatically starts playing (often blocked by browsers)
  • loop - Repeats the audio when finished
  • muted - Sets the audio output to muted by default
  • preload - Specifies how the audio should be loaded (auto/metadata/none)

HTML Tables

Tables are used to organize data into rows and columns. Here's a sample table structure:

<table> <thead> <tr> <th>Header 1</th> <th>Header 2</th> </tr> </thead> <tbody> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </tbody> </table>

Example: HTML Table Tags

Tag Description Example
<table> Defines a table <table>...</table>
<thead> Groups header content <thead><tr><th>...</th></tr></thead>
<tbody> Groups body content <tbody><tr><td>...</td></tr></tbody>
<tr> Defines a row <tr><td>...</td></tr>
<th> Defines a header cell <th>Header</th>
<td> Defines a data cell <td>Data</td>
